
# BSA

rust library and cli tool for creating and extracting Bethesda Softworks Archives.

## Support

|                                                    | Read | Write |
| -------------------------------------------------- | ---- | ----- |
| 100 (Tes3 Morrowind)                               | (x)  | (x)   |
| 103 (Tes4 Oblivion)                                | (x)  | (x)   |
| 104 (Tes5 Skyrim, Fallout: 4, Fallout: New Vegas)  | (x)  | (x)   |
| 105 (Tes5 Skyrim SE)                               | (/)  | (/)   |
| 105 (Fallout 4, Fallout 76)                        | (x)  | (x)   |
